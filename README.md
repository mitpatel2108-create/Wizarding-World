# 🪄 Wizarding World

An immersive magical web experience inspired by the world of fantasy magic, ancient castles, dark arts, legendary wizards, enchanted maps, and magical storytelling.

## ✨ Features

* 🏰 Stunning magical world inspired UI
* 🌌 Animated fantasy atmosphere and effects
* 🧙 Character exploration system
* 🗺️ Interactive magical locations
* 🪄 Dark arts and spell themed sections
* 📱 Fully responsive modern design
* ⚡ Fast performance using Vite
* 🎨 Beautiful UI components and animations
* ☁️ Cloudflare deployment ready

---

# 🚀 Tech Stack

## Frontend

* React
* TypeScript
* Vite
* Tailwind CSS
* TanStack Start

## UI & Styling

* Radix UI
* Framer Motion
* Lucide Icons
* Custom animations and magical effects

## Backend & Services

* Supabase
* Cloudflare Workers
* Cloudflare Pages

# 📂 Project Structure

```bash
src/
 ├── assets/          # Images and media
 ├── components/      # Reusable UI components
 ├── data/            # Static data and content
 ├── hooks/           # Custom React hooks
 ├── integrations/    # External integrations
 ├── lib/             # Utility functions
 ├── routes/          # Application routes/pages
 ├── server.ts        # Server entry
 └── start.ts         # App entry point
```

# ⚙️ Installation

Clone the repository:

```bash
git clone https://github.com/mitpatel2108-create/Wizarding-World.git
```

Go into the project folder:

```bash
cd Wizarding-World
```

Install dependencies:

```bash
npm install
```

---

# ▶️ Run Locally

Start the development server:

```bash
npm run dev
```

Open:

```bash
http://localhost:8080
```

---

# ☁️ Deployment

This project is configured for:

* Cloudflare Pages
* Cloudflare Workers

Build command:

```bash
bun run build
```

Deploy command:

```bash
npx wrangler deploy
```
